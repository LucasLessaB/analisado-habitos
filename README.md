# 📊 Analisador de Hábitos Pessoais

Projeto desenvolvido em Python para registrar e analisar hábitos diários como sono, consumo de água e exercícios.

## 💡 Funcionalidades

- Registrar hábitos diários (sono, água e exercício)
- Salvar dados automaticamente em arquivo JSON
- Calcular estatísticas (média, máximo e mínimo)
- Exibir relatório do dia com barra de progresso visual
- Detectar sequências consecutivas de dias (streak)
- Exportar relatório completo em arquivo .txt

## 🛠️ Tecnologias utilizadas

- Python 3.14
- Módulos: `json`, `os`, `statistics`, `datetime`
